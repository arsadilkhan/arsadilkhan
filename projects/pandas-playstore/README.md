# Play Store Data Cleaning (Pandas)

## Objective
Clean and prepare the raw `playstore.csv` dataset for downstream analytics and BI visualization.

## Dataset
- Google Play Store apps dataset
- Issues addressed: missing values, duplicates, non-numeric fields, inconsistent categories

## Work Performed
- Removed duplicate records
- Cleaned and standardized price, reviews, installs
- Handled missing values
- Normalized categorical columns
- Exported a clean, BI-ready CSV

## Tech Stack
- Python, Pandas, Jupyter

## Output
- `playstore_cleaned.csv`
- `playstore_cleaning.ipynb`

## Next Step
Dataset is used for Power BI visualization.
